# 期中作业

## 任务 1：微调在 ImageNet 上预训练的卷积神经网络实现宠物识别

### 基本要求

1. **数据集与 Baseline**  
   在 102 Category Flower Dataset 数据集上，修改现有的 CNN 架构（ResNet-18 或 ResNet-34）的输出层作为 Baseline 进行训练和测试。训练时使用在 ImageNet 上预训练得到的网络参数进行初始化，从零开始训练新的输出层，并对其余参数使用较小的学习率进行微调。

2. **超参数分析**  
   观察不同的超参数，如训练步数、学习率，及其不同组合带来的影响，并尽可能提升模型性能。

3. **预训练消融实验**  
   与仅使用 102 Category Flower Dataset 数据集从随机初始化的网络参数开始训练得到的结果进行对比，观察预训练带来的提升。

4. **引入注意力机制**  
   在 Baseline 模型的基础上，手动修改网络结构，加入注意力模块（如 SE-block、CBAM），或者直接使用轻量级的 Vision Transformer（如 ViT-Tiny、Swin-T），进行训练并与 Baseline 对比模型性能（Accuracy）。

## 任务 2：场景目标检测与视频多目标跟踪

### 基本要求

1. 使用 Road Vehicle Images Dataset 数据集，微调训练 YOLOv8 或同类现代单阶段模型，得到你的专属检测模型。

2. **视频流检测与多目标跟踪**  
   准备一段时长 10-30 秒的测试视频（可以是自己手机拍摄的校园/路口视频等）。利用你训练好的模型，结合多目标跟踪算法（如 YOLOv8 内置的 tracking），对视频进行逐帧推理。要求不仅输出 Bounding Box 与类别，还必须为每个同一目标分配稳定的跟踪 ID（Tracking ID）。

3. **遮挡与 ID 跳变分析**  
   多目标跟踪中最具挑战性的问题是物体遮挡。请在你的测试视频中，挑选一个发生目标遮挡或密集交汇的片段，截取连续的 3-4 帧画面进行可视化展示。结合课上所学理论，详细分析跟踪算法在面对此场景时，是成功维持了原有 ID，还是发生了目标丢失 / ID 跳变，并简述其原理或原因。

4. **越线计数**  
   在视频画面中设定一条虚拟的线，利用检测框中心的坐标与 Tracking ID 的连续性，编写简单的逻辑判断，统计并显示视频播放期间“跨越该线的物体总数”。

## 任务 3：从零搭建与损失函数工程：图像分割模型的像素级训练

### 基本要求

1. **网络搭建**  
   请勿使用任何预训练权重，使用深度学习框架的基础 API，从零开始手写搭建一个经典的语义分割网络 U-Net，必须包含完整的下采样编码器、上采样解码器以及特征拼接 Skip Connection 的逻辑。

2. **轻量级数据集训练**  
   为了在无预训练的情况下保证收敛速度，使用轻量级的分割数据集 Stanford Background Dataset。使用随机初始化的权重开始训练。

3. **损失函数工程**  
   图像分割任务常面临严重的前景 / 背景像素不平衡问题。请手动编写代码实现 Dice Loss。分别使用以下三种损失配置训练模型，并对比它们在验证集上的 mIoU 表现：

   - 仅使用标准的交叉熵损失（Cross-Entropy Loss）
   - 仅使用手动实现的 Dice Loss
   - 组合损失（Cross-Entropy Loss + Dice Loss）
